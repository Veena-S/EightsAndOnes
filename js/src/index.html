Eights & Ones

Eights & Ones is a two- or four-player board game from India. This game is similar to Chowka Bara or Ashta Chamma, with variations.
This game is an example of a ‚Äúfully observable‚Äù system that has an element of chance introduced by the roll of special dice and an element of strategy (the strategy being the pawn the player decides to move after the roll of the dice).

Eights & Ones normally has 5x5 square board and four players. But the number of squares can be increased to any size NxN, where N is an odd number. Each player will have 4 pawns or game tokens. Goal of a player in the game is to bring all the 4 pawns into the centre cell.

Game Rules

  1. A player can enter the game board through the home cell only if he manages to get point = 1 or point = 4
  2. If a player gets 4, he can either:
    • choose to enter all the 4 tokens into the home cell or
    • choose to start with only one token and move 3 cells forward
  3. Only at this point 4 can be split.
  4. At any other point of game, this splitting is not allowed.
  5. Once entered into the board, each player can throw the 4 dices when their turn comes
  6. Can move any of the tokens inside the board, to a cell by the number of points received.
  7. A cell can store multiple tokens of the same player.
  8. If a token comes to set in a cell where another player‚Äôs token is already present, old ones will be cut out by the current players token
  9. Rest/Safe locations
    • Each player has a home cell, which serves as the entry point to the game for that player, making 4 entry points alias safe cells
    • All these 4 entry points are safe points for the all the players i.e. tokens of different players can be stored in the safe cells
  10. Goal - A player has to make all the 4 tokens to reach the centre cell, which is the final destination
  11. Winner - First player who manages to bring all the 4 tokens at the final point
